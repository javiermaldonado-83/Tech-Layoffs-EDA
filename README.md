# Tech Industry Layoffs Analysis (2020–2025)

Exploratory Data Analysis of global tech layoffs using Python, Pandas, Matplotlib, and Seaborn.

## Overview
This project analyzes 2,412 layoff records across the tech industry from 2020 to 2025, 
exploring patterns by year, industry, country, company, and corporate stage.

## Key Findings
- 2022-2023 saw the largest layoff wave, driven by post-COVID over-hiring corrections
- USA accounts for the vast majority of global tech layoffs
- Consumer and Retail dominated early layoffs; by 2025 the wave shifted to core tech (Hardware, Software, IT Consulting)
- Post-IPO companies were the most affected, facing shareholder pressure to cut costs
- January, April, and November concentrate the most layoffs, aligned with fiscal planning cycles

## Tools & Libraries
- Python 3
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

Source: [Kaggle — Tech Layoffs Dataset](https://www.kaggle.com/)
